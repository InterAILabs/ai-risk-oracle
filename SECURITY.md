# Security Policy

InterAI Risk Oracle is hosted verification and trust infrastructure for
autonomous systems. Please report vulnerabilities privately.

## Reporting A Vulnerability

Contact security@interai.example with:

- affected endpoint, schema, SDK, or discovery document
- minimal reproduction steps without real secrets
- expected impact
- relevant request and response samples with credentials removed

Do not post secrets, exploit details, account identifiers, private receipts, or
production transaction data in public issues.

## Public Repository Scope

This repository intentionally excludes backend internals, billing systems,
private databases, signing secrets, scoring logic, and deployment configuration.
Reports about missing proprietary implementation code are out of scope.
