# Billing

InterAI Risk Oracle is a hosted service. Billing, metering, account balances,
payment processing, and ledger systems are operated by InterAILabs and are not
included in this public repository.

Public integrations should treat billing as an account-level service capability.
Use hosted pricing and onboarding metadata for the default self-serve path:

- https://ai-risk-oracle.fly.dev/pricing
- https://ai-risk-oracle.fly.dev/onboard

Use interailabs@gmail.com for support, security, enterprise access,
partnerships, or manual integration help.
